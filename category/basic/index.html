UI UX คืออะไร ?
เนื่องจากได้เฝ้ามองการพัฒนาแอ็ปบางตัวมาระยะหนึ่ง (ตัวแรกก็น่าจะตั้งแต่ปี 2555 กับอีกระบบที่กำลังพัฒนากันอยู่เป็นปีแล้วยังไม่เห็นรูปร่าง) เท่าที่มีโอกาสได้ร่วมฟังทางฝั่งผู้พัฒนาแอ็ปมีแต่คนพูดถึงการออกแบบตาม “ความต้องการ” (ซึ่งในที่นี่คือ TOR) ถึงแม้จะมีการเก็บ Requirement แต่ดูจากคำถามแล้ว
