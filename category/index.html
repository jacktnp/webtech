ทฤษฎีสี (Color Theory) ที่นักเรียนออกแบบต้องรู้ !!
ทฤษฎีสี (Theory of Color) หมายถึง ทฤษฎีของแม่สี ที่เป็นต้นกำเนิด ของการผสมสีเพื่อให้เกิดเป็นสี เพื่อนำไปใช้สร้างงานศิลปะหรืองานออกแบบแขนงต่าง ๆ

4 กฏการ วางองค์ประกอบ ทำตามนี้รับรองเป๊ะ
บางทีเราอาจจะคิดว่าเรามีกราฟิกที่ สวย สด งดงาม แต่มันจะดูแย่ลงทันทีถ้าหากเรา วางองค์ประกอบ พลาดไป มันทำให้งานที่คุณมั่นใจดูแย่ลงทันที ดังนั้นการว่างองค์ประกอบจึงเป็นเรื่องสำคัญสุดมากสำหรับนักออกแบบอย่างเราเพื่อทำให้งานออกแบบของเราออกมาสมบูรณ์ตามที่เราหวังไว้

ONE COLOR 3D DESIGN ออกแบบงานกราฟิกด้วยสีเดียว
แต่ช่วงนี้เรามักเห็นการนำเสนอผลิตภัณฑ์โดยใช้สีพื้นหลังสีเดียวกับตัวผลิตภัณฑ์มากขึ้น หรือเรียกว่า One Color 3D Design การนำเทคนิคสามมิติมาช่วยให้ผลิตภัณฑ์หรือวัตถุโดดเด่นออกมาจากพื้นหลังที่มีสีเดียวกันบวกกับการจัดแสงและเงายิ่งเพิ่มมิติให้ตัวงาน งานออกแบบที่ใช้สีเดียว
อ่านเพิ่มเติม
“TYPOGRAPHY STILL RULES” เทคนิคการใช้ตัวอักษรกับงานกราฟิก
เพราะตัวอักษรสำคัญกับงานกราฟิก นอกจากช่วยสื่อความหมายให้งานกราฟิกแล้ว เราสามารถนำตัวอักษรมาผสมผสานให้เข้ากับงานและสร้างความแปลกใหม่ให้กับงานกราฟิกได้อีกด้วย เราจึงแนะนำเทคนิคการสร้างสรรค์งานกราฟิกเจ๋ง ๆ ด้วยตัวอักษรด้วย Typography Still Rules 4 เทคนิคให้ได้เรียนรู้กัน
อ่านเพิ่มเติม
องค์ประกอบงานกราฟิก (Element of Design)
รูปร่าง เป็นองค์ประกอบต่อเนื่องมาจากเส้น เกิดจากการนำเส้นแบบต่าง ๆ มาต่อกันจนได้รูปร่าง 2 มิติที่มีความกว้างและความยาว (หรือความสูง) ในทางศิลปะจะแบ่งรูปร่างออกเป็น 2 แบบคือ รูปร่างที่คุ้นตา แบบที่เห็นแล้วรู้เลยว่านั่นคืออะไร
อ่านเพิ่มเติม
UI UX คืออะไร
UX สำคัญมากและเป็นหัวใจของการออกแบบ เนื่องจากได้เฝ้ามองการพัฒนาแอ็ปบางตัวมาระยะหนึ่ง (ตัวแรกก็น่าจะตั้งแต่ปี 2555 กับอีกระบบที่กำลังพัฒนากันอยู่เป็นปีแล้วยังไม่เห็นรูปร่าง) เท่าที่มีโอกาสได้ร่วมฟังทางฝั่งผู้พัฒนาแอ็ปมีแต่คนพูดถึงการออกแบบตาม “ความต้องการ” (ซึ่งในที่นี่คือ TOR)
อ่านเพิ่มเติม
ทฤษฎีเกสตอลท์สำหรับงานออกแบบ UI (Gestalt Theory for UI Design)
เกสตอลท์ (Gestalt) เป็นคำในภาษาเยอรมัน หมายถึง ‘รูปร่าง (Shape / Form)’ ซึ่งเป็นทฤษฎีที่ใช้อธิบายการรับรู้ภาพของมนุษย์ (Visual Perception)

เทคนิคการดีไซน์ปุ่มสำหรับ UX / UI Designer
ปุ่ม (Button) นับว่าเป็นชิ้นส่วนในการออกแบบเว็บไซต์พื้นฐานที่ทุกเว็บไซต์ ทุกแอพพลิเคชั่นจำเป็นต้องมี ไม่ว่าเราจะดีไซน์ User Interface (UI Design) หรือกำลังร่าง Wireframe (UX Design) ก็ตาม
อ่านเพิ่มเติม
จิตวิทยาพื้นฐานมนุษย์ที่คุณต้องรู้ ถ้าคิดจะออกแบบ UX
UX Design (User Experience) คือ การดีไซน์จากประสบการณ์ผู้ใช้ แปลว่าเราก็ต้องเข้าใจผู้ใช้ก่อนว่าเขาเป็นใครกัน ใช้อะไรกันบ้าง เคยชินกับอะไร ฟังเผินๆ UX เหมือนจะเป็นเรื่องที่ไม่มีหลักเกณฑ์

3 เคล็ดลับใช้ Infographic ทำตลาดอย่างมีประสิทธิภาพ
ไม่ว่าจะเป็นแบรนด์ใหญ่ หรือ SME ก็มีโอกาสได้รับความสนใจจากกลุ่มเป้าหมายได้ หากมีคอนเท้นต์ที่ดึงดูดและโดนใจ โดยเฉพาะในปัจจุบันที่สื่อออนไลน์ได้มีบทบาทกับการทำตลาดอย่างมาก คุณสามารถเลือกใช้เครื่องมือได้อย่างหลากหลาย เพื่อสร้าง Engagement ระหว่างแบรนด์กับผู้บริโภค และหนึ่งในคอนเท้นต์ที่ได้รับความนิยมต่อเนื่องมาหลายปีอย่าง Infographic

ทำ Infographic ด้วยตัวเอง แค่คลิกๆ ก็เสร็จแล้ว
วันนี้เรามีอีกหนึ่งเครื่องมือที่อยากจะแนะนำ นั่นก็คือ เว็บไซต์ที่ช่วยให้คุณสร้างแผนภูมิภาพ (Infographic) ด้วยสถิติใดๆ ก็ได้ที่คุณไม่ โดยแผนภูมินั้นมีให้เลือกทั้งแบบกราฟแท่ง กราฟเส้น เป็นรูปไอค่อน แผนที่ ข้อความ คลิปวิดีโอ และภาพทั่วไป
